# SnowTodo v0.1.0 🎉

首个正式版本发布！

## ✨ 新功能

### 核心功能
- **待办管理** - 创建、编辑、删除待办事项，支持优先级和截止日期
- **分类系统** - 自定义分类（工作/生活/学习等），支持分类筛选
- **标签管理** - 多标签标记，灵活组织任务
- **智能筛选** - 今日/全部/即将到期/已完成多视图切换

### 高级模块
- **🍅 番茄工作法** - 环形计时器 + 全局快捷键 + 中断记录
- **📊 数据仪表盘** - 纯 SVG 柱状图 + 环形图 + 周期汇总
- **💪 健康小助手** - 间隔/固定提醒 + 弹窗通知 + 番茄钟跳过
- **🤖 AI 智能助手** - OpenAI 兼容对话 + 任务上下文注入
- **⏰ 时间块视图** - 日程时间轴 64px/hour + 当前时间指示器

### 其他特性
- **多主题系统** - 6套内置主题 + 自定义主题 + 实时 CSS 变量切换
- **数据持久化** - SQLite 本地存储，支持导入导出
- **快捷键支持** - 全局快捷键快速操作

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/wxychenjian/snowtodo.git

# 安装依赖
cd snowtodo/app
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 📝 技术栈

- **Electron** - 跨平台桌面应用框架
- **React** - UI 组件库
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **SQLite (sql.js)** - 本地数据库
- **Zustand** - 状态管理
